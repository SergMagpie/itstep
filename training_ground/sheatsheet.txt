git log перегляд останніх комітів (локально)
git add --all - оновити репозиторій (локально)
git commit -m 'initial project version' - додати коміт до репозиторію (локально, після оновлення)
git commit -a -m 'initial project version' - додати коміт до репозиторію (локально, без необхідності команди git add)
